# Deno REST API
> Deno simple rest-api 
## Run
```
deno run --allow-net index.ts
```

## Routes
```
GET      /api/products
GET      /api/products/:id
POST     /api/products
PUT      /api/products/:id
DELETE   /api/products/:id
```

## Check 
```
 $ curl get localhost:5000/api/prouducts
```
